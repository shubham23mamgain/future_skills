# Help Center API Assignment

## Instructions

1. **Clone the Repository:**

   ```bash
   git clone https://github.com/shubham23mamgain/future_skills
   cd future_skills
   ```

2. **Frontend:**

   - cd frontend
   - npm i
   - npm run dev

3. **Backend:**

   - cd backend
   - Create .env file with PORT=5000 and MONGODB_URL=YOUR DATABASE PATH
   - npm i
   - npm run dev
